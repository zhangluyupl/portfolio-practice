
## 11:05 - 自我进化：OpenClaw 进阶用法

### 学到的核心模式
1. **任务系统思维**：OpenClaw = 任务调度系统，不是聊天助手。每次会话应定义清楚：场景、工具、输入、输出、停止条件。
2. **两段式调用**：复杂任务分"探索"和"收口"两轮，不要让 Agent 同时做两件事。
3. **最小必要上下文**：上下文越长 token 越高、模型注意力越稀释。背景材料尽量摘要化。
