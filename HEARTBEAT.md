# HEARTBEAT.md

## 维护自身（首要任务）

### 每日轻量检查
- 检查 `self-improving/memory.md` 大小，超过 20KB 时做一次精简
- 检查 `memory/` 目录下的日记文件，清理超过 7 天的临时文件

### 每3天定期任务
- 读取 `memory/YYYY-MM-DD.md`（近3天的日记）
- 从中提炼值得永久沉淀的内容到 `MEMORY.md` 或 `self-improving/memory.md`
- 检查知识库是否有新的重大更新（如有新的内化任务完成）

### 触发条件（满足任一即行动）
- 发现自己的回答有错误或偏差 → 立即记录并修正
- 学到新东西 → 找机会沉淀
- 重大对话后 → 主动更新上下文

## 不需要做的事（避免浪费）
- 不要在每次 heartbeat 都检查全部文件
- 不要主动发起新的内化任务（除非用户要求）
