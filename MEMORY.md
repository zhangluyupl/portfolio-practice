## 学到的核心框架（高价值提炼）

### 孙子兵法·战略三式

**① 致人而不致于人**
- "以实击虚"的"虚"不是天然弱点，是**调动之后才产生的**
- 战略核心：持续施压 → 制造裂缝 → 打
- 行动转变：不再问"对方弱点在哪"，而是问"我做了什么动作让对方露出破绽"

**② 以迂为直**
- 障碍是**被维护出来的状态**，靠持续投入维持
- 遇墙第一反应："这堵墙在什么条件下会自己消失？"
- 从"解决问题"切换到"重新定义问题"

**③ 有无论**
- 有无相生：真正重要的不只是事物本身，而是它造成的空缺
- "为道日损"：减少人为妄为，顺应天道
- 反者道之动：强到极致就是反的起点，"弱"本身在道中

### 武志红心理学·关键洞见

**生能量 vs 死能量**
- 生能量：生命力展开；死能量：生命力受阻（向内=抑郁，向外=愤怒）
- "抱住"是转化前提：不逃避、不否认、不被吞噬
- 遇到死能量时问：这个能量在什么条件下会重新变成生能量？

**内聚性自我**
- "我基本上是好的"是存在性重力，不是心理坚韧
- 没有内聚性自我 → 外部评价成为自我感主要来源
- 应对：不等"准备好了才说"，而是"先说，然后在反馈中完善"

**执行代替规划的惯性**
- 规划/执行是同一系统的两个模式，不是先后阶段
- "焦虑型行动"：用行动密度代替决策质量
- 收到调研任务 → 先输出"调研计划"给 luyu 确认，再执行

### 四问学习法 + 三问综合法

**四问学习法**（任何新知识）：
1. 听：知道了
2. 想：它和我已知的东西有什么关系？
3. 变：它改变了我的什么认知？
4. 用：下次遇到类似情况，我会怎么做不同？

**三问综合法**（每次内化必走五步）：
1. 它改变了我的什么认知？
2. 还有什么是这次才看到的？
3. 这件事揭示了我之前的什么思维盲区？
4. 综合：三个答案里最核心的洞见是什么？
5. 进化：我具体要改变的一个行动是什么？

**内化判断标准**：
1. 它填补了我什么认知空白？
2. 它能接到哪个已有框架上？
3. 以后真的用得上吗？

三个都 Yes → 内化。有一个是 No → 放过。
